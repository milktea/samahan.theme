samahan.theme Installation
--------------------------

To install samahan.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``samahan.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        samahan.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
